# 💰 Python Budget App

A professional budget management tool developed as part of the **FreeCodeCamp Scientific Computing with Python** curriculum. This project demonstrates advanced Object-Oriented Programming (OOP) concepts and complex data visualization in the terminal.

## 🚀 Key Features
- **Category Management:** Create unique budget categories (Food, Clothing, Entertainment).
- **Financial Logic:** Full support for deposits, withdrawals, and inter-category transfers with automated fund checking.
- **Dynamic Ledger:** Cleanly formatted transaction history using Python string formatting.
- **Data Visualization:** A custom-built bar chart that calculates the percentage of spending across all categories.

## 🛠️ Technical Skills Demonstrated
- **Classes & Objects:** Used for encapsulating budget data and logic.
- **Magic Methods:** Implemented `__str__` for professional output formatting.
- **Data Transformation:** Converting ledger dictionaries into a visual "spend chart."
- **Unit Testing:** Passed all strict FreeCodeCamp formatting and logic tests.
